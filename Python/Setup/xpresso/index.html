    構築手順

    手順は至ってシンプル。

    1. 作業用ディレクリを作成し、移動
    2. git clone https://github.com/CiscoTestAutomation/xpresso を実行し、リポジトリをクローンする
    3. ./setup.sh http://<xpresso を立てる server のURL>/
    4. docker-composepull　を実行
    5. docker-compose up -d を実行
    6. ブラウザにてhttp://localhostでxpresso にアクセスする

    Info

    上記手順3にてスクリプトではなく、手動で設定することで構成をカスタマイズすることができます。 詳しくはこちらに記載されています。
